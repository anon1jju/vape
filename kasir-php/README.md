# Kasir PHP JSON

## Cara Jalankan
1. Letakkan folder `kasir-php` di `htdocs` (XAMPP) atau server PHP.
2. Jalankan: `http://localhost/kasir-php`
3. Login default: **admin / admin123** atau **kasir / kasir123**

## Fitur
- Login + Role (admin/kasir)
- CRUD Kategori
- CRUD Produk + Stok
- CRUD User
- Transaksi Kasir (keranjang + diskon + pajak + pembayaran)
- Cetak Struk
- Laporan + Export CSV

## Catatan
Database memakai file JSON di folder `data/`.
